# Student Performance Analytics Dashboard

This project is an interactive Streamlit dashboard for analyzing student academic performance.

## Overview

The dashboard explores how demographic, study-related, and behavioral factors are associated with students' final grades.

## Features

- Upload student performance dataset
- Interactive filters by gender, internet access, and study time
- KPI cards for total students, average final grade, study level, and absences
- Visualizations for grade distribution, study time, previous grades, and alcohol consumption
- Summary insights for educational data analysis

## Tools Used

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Statsmodels

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
