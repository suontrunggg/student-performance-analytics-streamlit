# Student Performance Analytics Dashboard Using Streamlit

This project aims to build an interactive dashboard for analyzing student academic performance using Streamlit. The dashboard allows users to upload a student performance dataset and explore important academic indicators through filters, KPI cards, and visualizations.

The main purpose of this dashboard is to help users understand how different factors may relate to students’ final grades. Users can filter the data by gender, internet access, and study time, then observe changes in key metrics such as total number of students, average final grade, average study level, and average absences.

The dashboard also provides several visualizations, including the distribution of final grades, the relationship between study time and final grade, the correlation between previous grade and final grade, the effect of weekend alcohol consumption on performance, and the influence of parental education when available in the dataset.

The final output is a working Streamlit dashboard that presents student performance data in a clear and interactive way. This helps non-technical users quickly identify patterns and insights from the dataset without needing to write code.
